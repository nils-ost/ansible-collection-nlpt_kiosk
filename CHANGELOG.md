## v1\.0\.0

<a id="release-summary"></a>

### Release Summary

This is the first proper release of <code>nils_ost\.nlpt_kiosk</code> collection on 2026\-03\-24\.
This release contains everything that is needed to install an Kiosk\-Controller and configure Raspberry Pis to act as Kiosks\.

<a id="new-roles"></a>

### New Roles

- nils_ost\.nlpt_kiosk\.controller \- installs NLPT\-Kiosk\-Controller within docker\.
- nils_ost\.nlpt_kiosk\.rpi \- configures Raspberry Pi to be a Kiosk\, connected to NLPT\-Kiosk\-Controller\.
